### Database Commands

```terminal
$ createdb database-name
$ dropdb database-name

$ psql

=# \l  // list databases
=# \c database-name  // change database
=# \d table-name  // shows table info
=# \dt  // shows all the tables

=# table table-name;  // show all entries
=# select * table-name;  // show all entries

=# \q  // quit

```
